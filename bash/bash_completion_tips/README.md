# 命令解释

| 命令                | 作用                                                                                   |
|---------------------|----------------------------------------------------------------------------------------|
| bypy                | bypy是一个命令行上传下载百度云盘上的文件的python客户端                                 |
| ebook-convert       | calibre 提供的命令行工具, 用于转换电纸书格式, 支持pdf, epub等                          |
| jq                  | json支持命令行                                                                         |
| nmap                | 扫描局域网下那些ip开放了目标端口                                                       |
| nmcli               | linux命令行配置wifi                                                                    |
| ocrmypdf            | ocrmypdf是一个扫描pdf文件添加 OCR 文本层的python工具                                   |
| pandoc              | 电子书文件格式转换工具, 比如epub转pdf                                                  |
| rsync               | 它可以在本地计算机与远程计算机之间，或者两个本地目录之间同步文件                       |
| sed                 | 替换字符串命令行                                                                       |
| sox                 | 音频处理命令行                                                                         |
| update-alternatives | update-alternatives 是一个切换命令行版本的工具, 本质上是管理某个软链接指向那个那个文件 |
