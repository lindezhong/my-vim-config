基础设施层、存储库接口实现、请求外部服务(防腐层可以在这里)。这是应用程序的基建部分, 提供一些底层能力。

对于请求外部服务, 它的接口可定义在其它层(比如接口定义在domian, application上), 但实现在infrastructure, 顺带的可以在这里实现防腐层, 对于每个外部服务的实现来说就是一个package

除了三个垂直层之外，还有基础设施层。它以不同的方式支持所有三个层，从而促进各层之间的通信。 简而言之，基础设施层包含独立于应用程序的所有组件：外部库、数据库引擎、应用服务器、消息传递后端等等。

此外，我们将连接其他层和基础设施层的代码和配置文件视为基础设施层的一部分。 例如，在持久层方面，数据库模式定义、Hibernate 配置和映射文件以及存储库接口的实现都属于基础设施层。
